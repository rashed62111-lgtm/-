# موقع محل علي عيد لمواد البناء

موقع ويب مجاني ومتجاوب لمحل مواد البناء يتيح للعملاء تقديم الشكاوى وطلبات الدعم الفني والنصائح.

## المميزات

✅ **تصميم متجاوب**: يعمل بشكل ممتاز على الهاتف واللاب توب  
✅ **تغيير الألوان التلقائي**: يتغير لون الموقع كل 3 ثواني  
✅ **نماذج التواصل**: شكاوى، دعم فني، ونصائح  
✅ **لوحة إدارة محمية**: للتحكم في الطلبات الواردة  
✅ **إرسال الإيميلات**: عبر EmailJS (مجاني)  
✅ **حفظ محلي**: جميع الطلبات تُحفظ محلياً  

## كيفية الاستخدام

### 1. فتح الموقع
- افتح ملف `index.html` في المتصفح
- أو ارفع الملفات على استضافة مجانية (مثل GitHub Pages, Netlify, Vercel)

### 2. إعداد إرسال الإيميلات (اختياري)

لإرسال الإيميلات عند وصول طلبات جديدة:

1. سجل في [EmailJS](https://www.emailjs.com/) (مجاني)
2. أنشئ Service و Template
3. افتح `script.js` وحدّث:
   - `YOUR_PUBLIC_KEY`: Public Key من EmailJS
   - `YOUR_SERVICE_ID`: Service ID
   - `YOUR_TEMPLATE_ID`: Template ID

**ملاحظة**: حتى بدون إعداد EmailJS، الطلبات تُحفظ محلياً ويمكن رؤيتها في لوحة الإدارة.

### 3. إعداد لوحة الإدارة

افتح `admin.js` وغيّر معلومات تسجيل الدخول:

```javascript
const ADMIN_CREDENTIALS = {
    username: 'admin',        // غيّر اسم المستخدم
    password: 'admin123'      // غيّر كلمة المرور
};
```

**معلومات الدخول الافتراضية:**
- اسم المستخدم: `admin`
- كلمة المرور: `admin123`

⚠️ **مهم**: غيّر كلمة المرور قبل نشر الموقع!

### 4. الوصول للوحة الإدارة

- من الصفحة الرئيسية، اضغط على "لوحة الإدارة"
- أو افتح `admin.html` مباشرة
- أدخل اسم المستخدم وكلمة المرور

## الملفات

- `index.html` - الصفحة الرئيسية
- `style.css` - التصميم والألوان
- `script.js` - الوظائف الرئيسية
- `admin.html` - لوحة الإدارة
- `admin.js` - وظائف لوحة الإدارة

## النشر المجاني

يمكنك نشر الموقع مجاناً على:

- **GitHub Pages**: ارفع الملفات على GitHub وفعّل Pages
- **Netlify**: اسحب وأفلت الملفات على Netlify
- **Vercel**: ارفع الملفات على Vercel
- **Firebase Hosting**: استخدم Firebase Hosting المجاني
